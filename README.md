### ESP32 Room Light Control

This project demonstrates how to use an **ESP32** to control a room light over Wi-Fi.

#### Hardware Setup

* Connect the **ESP32** to the room's light switch (relay/interrupter circuit).
* Power on the ESP32.

#### How It Works

1. The ESP32 creates a Wi-Fi access point.
2. Connect any Wi-Fi-enabled device (smartphone, tablet, or laptop) to the ESP32's network.
3. Open the ESP32 web interface in your browser.
4. Use the control page to toggle the room light **ON** or **OFF**.

#### Features

* Remote light control over Wi-Fi
* No Internet connection required
* Simple web-based interface
* Compatible with any device that has a web browser

> **Note:** Ensure the relay and wiring are rated for your mains voltage and follow appropriate electrical safety practices when connecting the ESP32 to household lighting.
